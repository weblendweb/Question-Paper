4. None of the above.

HINT:  The resonance hybrid should have lower energy than any of the contributing structure.

Explanation:

Conditions of resonance are: 

1. The relative position of all the atoms must be same in each resonating structures.

2. Number of paired and unpaired electrons in each resonating structure must be same so that there is a continuous change from one bond type to another bond  type may occur.

3. Contributing structures should have almost equal energy .

4. Like charges should be greatly separated and unlikely charges should be placed together.

6. The resonance hybrid should have lower energy than any of the contributing structure.

An amphiprotic (can accept and give proton) ion is-

1. HPO−3HPO3-

2. H2PO−2H2PO2-

3. H3PO4H3PO4

4. H2PO−4H2PO4-
Ans. 4
Sol.

Hint: Accept and give up a proton.

An amphiprotic substance can both donate or accept a proton.

HPO−3HPO3- is a conjugate base of H2PO3H2PO3. It has only one acidic hydrogen. Thus, HPO−3HPO3- can only act as a base, not an acid.

H2PO−2H2PO2- is a conjugate base of H3PO2H3PO2. It has only one acidic hydrogen. Thus, H2PO−2H2PO2-  can only act as a base, not an acid.
